# Data Provenance — Pluviometrics Hub

The hub is a static landing page. It ships:

- Brand logos and copy (first-party).
- Outbound `<a href>` links to Pluviometrics products on other domains.

It does not ship, embed, or fetch any third-party data.

For each linked product's data provenance, see that product's
`DATA_PROVENANCE.md`:

- Stormgauge: `pluvio-stormgauge/DATA_PROVENANCE.md`
- Stormgrid: `stormgrid/DATA_PROVENANCE.md` (or `pluvio-stormgrid/DATA_PROVENANCE.md` after the org rename)
- Atmos: published when Atmos repo is consolidated
- NBC legacy: NBC-internal — provenance documented by Northern Beaches Council
